# 🧠 AI-Based Multi-Class Detection of Fetal Brain Abnormalities Using CNN

## 📌 Project Overview

This project is a deep learning-based web application developed to detect fetal brain abnormalities from prenatal ultrasound images using a Convolutional Neural Network (CNN).

The application allows users to upload an ultrasound image and predicts the type of fetal brain abnormality along with the confidence score.

---

## 🚀 Features

- Upload fetal brain ultrasound images
- Automatic image preprocessing
- CNN-based disease prediction
- Displays predicted disease
- Displays confidence score
- User-friendly Flask web interface
- Shows uploaded image along with prediction

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/Fetal_Brain_Abnormality_Detection.git
```

Install the required libraries:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000
```
